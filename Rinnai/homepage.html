Project Overview

The aim of this project is to capture record the test data electronically and automate the testing process. With the new systems in place, we would be able to look up the test data for the appliance more easily & will be able to perform analysis on the test data

There will be 3 phases to this project. The frist phase is to convert the current paper-based data recording systems into eletronic version. In the second phase we hope to have system in place which will capture test data straight off the test equipment.eg. Micro amp meters, flow meters, maometeers & etc.

The third phase is to automate the testing process ie. pressure setting, test data recording & analysing and then inform the user of the test result. Critical tests such as the PAT test & gas leak detection test will still be performed by the test operator. These would thereby increase productivity and data accuracy.
